Introduction

The freeCodeCamp Responsive Web Design course initiates learners into the pivotal realm of responsive design, exploring its critical role in crafting websites that seamlessly adapt to diverse screen sizes and devices. This foundational module sets the stage for a deep dive into HTML and CSS.

Am Marvin and this is my experience and journey through the first course with freeCodeCamp.

Introduction to html

Responsive Web Design Certification, here we learnt the languages that developers use to build webpages: HTML (Hypertext Markup Language) for content, and CSS (Cascading Style Sheets) for design.

First, we built a cat photo app to learn the basics of HTML and CSS. Later, we learnt modern techniques like CSS variables by building a penguin, and best practices for accessibility by building a quiz site.

Here we go introduced to topics like;

  • HTML Structure: Understand the building blocks of HTML, including tags, elements, and attributes.
  • Document Anatomy: Learn about the structure of HTML documents, differentiating between the head and body sections.
  • Navigation Basics: Create functional hyperlinks to enhance website navigation.
<!DOCTYPE html> <html> <head> </head> <body> </body> </html>
Introduction to CSS

CSS tells the browser how to display your webpage. You can use CSS to set the color, font, size, and other aspects of HTML elements.

In this course, we learnt CSS by designing a menu page for a cafe webpage.

Here, we were introduced to topics like;

  • Styling Essentials: Grasp the fundamentals of CSS, covering selectors, properties, and values.
  • Integration with HTML: Link CSS stylesheets to HTML documents for cohesive styling.
  • Text and Background Styling: Apply CSS to stylize text, backgrounds, and borders.
body { font-family: Arial, sans-serif; background-color: #f2f2f2;
color: #333; border-bottom: 2px solid #555; }
CSS Colors

Diversified my palette with RGB, HEX, and HSL color notations. Apply these color styles to text, backgrounds, and borders, adding vibrancy and depth to your web design.

I learnt all the different notations of CSS color i.e. RGB, HEX and HSL (shown in the field below) through building a set of color makers.

/* RGB */ div { background-color: rgb(255, 0, 0);
/* HEX */ color: #00ff00;
/* HSL */ border: 2px solid hls(120, 100%, 50%); }
CSS Box Model

Understanding the Box Model, Dissect the box model concept, unraveling the roles of margins, borders, padding, and content. Master the art of layout manipulation through strategic utilization of the box model.

This was through building a Rothko-style rectangular art pieces.

In this course, we used typography to build a nutrition label webpage. We learnt how to style text, adjust line height, and position your text using CSS.

Pseudo Selectors

Enhancing user interaction elevates user engagement by exploring pseudo-classes like :hover and :focus was a major area of focus in this section. Discovering the power of the :nth-child selector for dynamic styling.

CSS Grid

We acquired proficiency in using CSS Grid to design complex and responsive layouts.

By building a magazine, we learnt to create grid structures and control the placement of items within the grid. The example code illustrates defining a grid layout with three columns and a gap between them.

.container { display: grid; grid-template-columns: repeat(3, 1fr); grid-gap: 10px; }

CSS transforms allow for the manipulation of an element's size, shape, and orientation. Students learn about transformations like rotate, scale, and translate. The example code shows rotating a box element by 45 degrees.
